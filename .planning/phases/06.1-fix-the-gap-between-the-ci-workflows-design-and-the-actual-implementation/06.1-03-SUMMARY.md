## Accomplishments
- PublishVersionView validates semver format, uniqueness, ordering (> latest), regenerates manifest with version header, transitions draft to authorized
- RevokeVersionView clears manifest content and sets revoked status with revoked_by/revoked_at tracking
- DiscardDraftView and SuggestVersionView for draft management and version suggestion
- Workflow detail page converted from stacked layout to tabbed layout (Steps, Versions, Manifest, Services)
- Version history tab shows all versions with status badges, truncated hashes, author, published date, and revoke actions
- Publish modal collects version number (pre-populated with suggested next patch) and optional changelog
- Draft banner appears when unpublished draft exists with Publish and Discard actions
