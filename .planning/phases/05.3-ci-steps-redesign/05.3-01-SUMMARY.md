# Phase 5.3 Plan 01: CI Engine Abstraction Foundation Summary

Plugin-based CI capability mixin with engine-agnostic step discovery and model engine fields.

## What Was Done

### Task 1: CICapableMixin and GitHubPlugin Implementation
- Added `CICapableMixin` class to `plugins/base.py` defining the CI engine contract: `engine_name`, `engine_display_name`, `engine_file_name`, `parse_step_file`, `generate_manifest`, `manifest_path`
- Added `get_ci_plugin_for_engine()` and `get_available_engines()` helper functions
- Made `GitHubPlugin` inherit from both `CICapableMixin` and `BasePlugin`
- Implemented all CICapableMixin methods on GitHubPlugin including `generate_manifest` (moved from `core/ci_manifest.py`)
- Added `"ci"` to GitHubPlugin capabilities list

### Task 2: Engine Fields and core/ci_steps.py
- Added `engine` CharField (default `"github_actions"`) to `StepsRepository` model
- Added `engine` CharField (default `"github_actions"`) to `CIStep` model
- Generated and applied migration 0014
- Created `core/ci_steps.py` with `discover_steps()` (walks repo at any depth, skips hidden dirs, returns parsed YAML) and `parse_runtimes_yml()` (moved from git_utils.py)

### Task 3: Clean core/git_utils.py
- Removed `scan_ci_steps` function (replaced by `discover_steps` + plugin `parse_step_file`)
- Removed `parse_runtimes_yml` function (moved to `core/ci_steps.py`)
- Updated `core/tasks.py` to use new engine-agnostic discovery: `repository.engine` -> `get_ci_plugin_for_engine` -> `discover_steps` -> `parse_step_file`

## Decisions Made

| Decision | Rationale |
|----------|-----------|
| CICapableMixin before BasePlugin in MRO | Standard Python mixin ordering; mixin first, base class second |
| Helper functions in plugins/base.py not ci_steps.py | They query PluginRegistry which lives in base.py; avoids cross-module dependency |
| discover_steps returns raw directory_path | Engine-agnostic; callers use os.path.basename for directory_name |
| Lazy yaml import in generate_manifest | Avoid adding yaml to plugin module top-level imports |
