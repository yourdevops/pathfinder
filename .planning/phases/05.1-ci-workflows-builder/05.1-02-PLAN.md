  <action>
    **In core/git_utils.py, add two new functions:**

    1. `parse_runtimes_yml(repo_path: str) -> dict`:
       - Read `runtimes.yml` at repo root
       - Return dict mapping family name to list of version strings
       - Handle both `{family: {versions: [...]}}` and `{family: [...]}` formats
       - Return empty dict if file not found
       - Use `yaml.safe_load()`
       - Convert all version values to strings (in case YAML parses "3.11" as float)

    2. `scan_ci_steps(repo_path: str) -> list`:
       - Walk `ci-steps/` directory at repo root
       - For each subdirectory containing `action.yml` (or `action.yaml` fallback):
         - Parse with `yaml.safe_load()`
         - Extract `name`, `description`, `inputs` from top level
         - Extract `phase`, `runtimes` (as runtime_constraints), `tags`, `produces` from `x-pathfinder` block
         - Store full parsed YAML as `raw_metadata`
       - Return sorted list of dicts (sorted by directory name)
       - Return empty list if `ci-steps/` directory not found
       - Log warnings for directories without action.yml

    **In core/tasks.py, add:**

    `scan_steps_repository(repository_id: int) -> dict` as a `@task(queue_name='steps_scan')`:
    1. Import StepsRepository, RuntimeFamily, CIStep inside function (avoid circular imports)
    2. Get repository record, set `scan_status='scanning'`
    3. Get connection if set, build authenticated URL using `build_authenticated_git_url()`
    4. Clone repo using `clone_repo_shallow()` (reuse existing pattern from `sync_blueprint`)
    5. Call `parse_runtimes_yml()` on the cloned repo path
    6. For each runtime family: create/update `RuntimeFamily` record (use `update_or_create` on repository+name)
    7. Call `scan_ci_steps()` on the cloned repo path
    8. Get HEAD commit SHA from the cloned repo
    9. For each step: create/update `CIStep` record (use `update_or_create` on repository+directory_name)
       - Set all fields from the scan result dict
       - Set `commit_sha` to HEAD SHA
    10. Delete any CIStep records for this repository whose directory_name is NOT in the scan results (removed steps)
    11. Delete any RuntimeFamily records for this repository whose name is NOT in the parsed runtimes (removed families)
    12. Set `scan_status='scanned'`, `last_scanned_at=timezone.now()`, `scan_error=''`
    13. Cleanup repo in finally block
    14. On exception: set `scan_status='error'`, `scan_error=str(e)`
    15. Return dict with counts: `{"steps": N, "runtimes": M}`
  </action>
