Create core/git_utils.py with the following helper functions:

**Helper function: parse_git_url(url)**
- Parse any Git URL to extract host, owner, repo
- Support formats:
  - https://github.com/owner/repo
  - https://github.com/owner/repo.git
  - https://gitlab.com/owner/repo
  - git@github.com:owner/repo.git
  - git@gitlab.com:owner/repo.git
- Return dict with: host, owner, repo, or None if invalid
- This is SCM-agnostic (works with GitHub, GitLab, Bitbucket, etc.)

**Helper function: build_authenticated_git_url(git_url, connection=None)**
- Given a git_url and optional IntegrationConnection, return URL with embedded credentials
- If connection is None: return original URL (public access)
- If connection exists:
  - Get decrypted config via connection.get_config()
  - For GitHub connections with auth_type='token': use PAT as password
    - Format: https://{personal_token}@github.com/owner/repo.git
  - For GitHub App connections: get installation token via plugin._get_github_client_app()
    - This requires calling plugin to get a fresh installation token
    - Format: https://x-access-token:{installation_token}@github.com/owner/repo.git
  - Generic pattern: https://{user}:{token}@{host}/{owner}/{repo}.git
- Handle SSH URLs by converting to HTTPS with auth
- Return authenticated URL string

**Helper function: clone_repo_shallow(git_url, branch='main', auth_url=None, depth=1)**
- Import tempfile, git (GitPython)
- Create temp directory via tempfile.mkdtemp(prefix='ssp_blueprint_')
- Use git.Repo.clone_from(auth_url or git_url, temp_dir, depth=depth, branch=branch, single_branch=True)
- Return tuple: (repo_object, temp_dir_path)
- On error: clean up temp dir and re-raise

**Helper function: read_manifest_from_repo(repo_path)**
- Try to read 'ssp-template.yaml' from repo_path root
- If not found, try 'pathfinder-template.yaml'
- If neither found: raise FileNotFoundError("Manifest file not found")
- Parse with yaml.safe_load()
- Return parsed dict

**Helper function: list_tags_from_repo(repo)**
- Given a git.Repo object, list all tags
- Return list of dicts: [{'name': tag.name, 'commit_sha': tag.commit.hexsha}, ...]
- Sort by tag name

**Helper function: cleanup_repo(repo, temp_dir)**
- Close repo object
- Remove temp directory via shutil.rmtree(temp_dir, ignore_errors=True)

**Helper function: compute_version_sort_key(major, minor, patch, prerelease)**
- Return sortable string key for version ordering
- Format: f'{major:05d}.{minor:05d}.{patch:05d}.{prerelease or "zzzz"}'
- This ensures: 1.0.0 > 1.0.0-rc.1 > 1.0.0-beta.1

**Helper function: parse_version_tag(tag_name)**
- Strip leading 'v' or 'V'
- Use semver.Version.parse(version_str)
- Return dict with: major, minor, patch, prerelease, is_prerelease, sort_key
- On ValueError (non-semver tag): return major=0, minor=0, patch=0, prerelease=version_str, is_prerelease=True

Add to core/tasks.py the sync_blueprint task:

**Task: sync_blueprint(blueprint_id: int)**
- Queue name: 'blueprint_sync'
- Import from core.git_utils: clone_repo_shallow, read_manifest_from_repo, list_tags_from_repo, cleanup_repo, build_authenticated_git_url, parse_version_tag
- Get Blueprint by id, return error if not found
- Mark sync_status='syncing', clear sync_error
- Build authenticated Git URL:
  - If blueprint.connection exists: call build_authenticated_git_url(blueprint.git_url, blueprint.connection)
  - Otherwise: use blueprint.git_url directly (public repo)
- Clone repo with GitPython:
  - repo, temp_dir = clone_repo_shallow(blueprint.git_url, blueprint.default_branch, auth_url)
- Read manifest from local file:
  - manifest = read_manifest_from_repo(temp_dir)
- Update blueprint fields from manifest:
  - name: manifest.get('name', '')
  - description: manifest.get('description', '')
  - tags: manifest.get('tags', [])
  - ci_plugin: manifest.get('ci', {}).get('type', '')
  - deploy_plugin: manifest.get('deploy', {}).get('required_plugins', [None])[0] or manifest.get('deploy', {}).get('type', '')
  - manifest: manifest (full dict)
- List all git tags from repo:
  - tags = list_tags_from_repo(repo)
- For each tag:
  - Parse version info with parse_version_tag
  - Update or create BlueprintVersion record
- Remove versions for deleted tags (tags no longer in repo)
- Clean up temp directory: cleanup_repo(repo, temp_dir)
- Mark sync_status='synced', set last_synced_at=timezone.now()
- On exception: clean up temp dir, mark sync_status='error', sync_error=str(e)
- Return result dict with status, name, versions count

IMPORTANT: All Git operations use GitPython (not GitHub API). This enables support for any Git-compatible SCM.
