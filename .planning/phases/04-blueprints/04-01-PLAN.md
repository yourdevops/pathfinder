Add to core/tasks.py the following functions and task:

**Helper function: parse_github_url(url)**
- Parse GitHub URL to extract owner/repo
- Support formats:
  - https://github.com/owner/repo
  - https://github.com/owner/repo.git
  - git@github.com:owner/repo.git
- Return (owner, repo) tuple or None if invalid

**Helper function: compute_version_sort_key(major, minor, patch, prerelease)**
- Return sortable string key for version ordering
- Format: f'{major:05d}.{minor:05d}.{patch:05d}.{prerelease or "zzzz"}'
- This ensures: 1.0.0 > 1.0.0-rc.1 > 1.0.0-beta.1

**Helper function: parse_version_tag(tag_name)**
- Strip leading 'v' or 'V'
- Use semver.Version.parse(version_str)
- Return dict with: major, minor, patch, prerelease, is_prerelease, sort_key
- On ValueError (non-semver tag): return major=0, minor=0, patch=0, prerelease=version_str, is_prerelease=True

**Task: sync_blueprint(blueprint_id: int)**
- Queue name: 'blueprint_sync'
- Get Blueprint by id, return error if not found
- Mark sync_status='syncing', clear sync_error
- Get or auto-detect GitHub connection (find first active github connection if blueprint.connection is None)
- Get GitHub plugin from registry
- Get authenticated client via plugin._get_github_client(config)
- Parse git_url to get owner/repo
- Fetch ssp-template.yaml (try both 'ssp-template.yaml' and 'devssp-template.yaml')
- Parse manifest with yaml.safe_load (import yaml at top)
- Update blueprint fields from manifest:
  - name: manifest.get('name', '')
  - description: manifest.get('description', '')
  - tags: manifest.get('tags', [])
  - ci_plugin: manifest.get('ci', {}).get('type', '')
  - deploy_plugin: manifest.get('deploy', {}).get('required_plugins', [None])[0] or manifest.get('deploy', {}).get('type', '')
  - manifest: manifest (full dict)
- Fetch all git tags via repo.get_tags()
- For each tag:
  - Parse version info with parse_version_tag
  - Update or create BlueprintVersion record
- Remove versions for deleted tags
- Mark sync_status='synced', set last_synced_at=timezone.now()
- On exception: mark sync_status='error', sync_error=str(e)
- Return result dict with status, name, versions count
