    1. Install Python dependency: `uv add channels==4.3.2`

    2. Install npm dependency: `cd theme/static_src && npm install htmx-ext-ws@2.0.3`

    3. Update `theme/static_src/copy-vendor.js` -- add entry to the `files` array:
       `['node_modules/htmx-ext-ws/dist/ws.min.js', 'htmx-ext-ws.min.js']`
       Then run: `cd theme/static_src && node copy-vendor.js` to copy the file.

    4. Update `pathfinder/settings.py`:
       - Add `"channels"` to `INSTALLED_APPS` (after `"formtools"`, before `"tailwind"`)
       - Update `SECURE_CSP` to add `connect-src` directive:
         `"connect-src": [CSP.SELF, "ws:", "wss:"],`
       - CRITICAL: CSP `'self'` does NOT match ws:// across browsers -- must explicitly add ws: and wss:

    5. Update `pathfinder/asgi.py` to use ProtocolTypeRouter:
       ```python
       import os
       from channels.auth import AuthMiddlewareStack
       from channels.routing import ProtocolTypeRouter, URLRouter
       from channels.security.websocket import AllowedHostsOriginValidator
       from django.core.asgi import get_asgi_application

       os.environ.setdefault("DJANGO_SETTINGS_MODULE", "pathfinder.settings")

       # MUST call get_asgi_application() before importing consumers (AppRegistryNotReady otherwise)
       django_asgi_app = get_asgi_application()

       from core.routing import websocket_urlpatterns  # noqa: E402

       application = ProtocolTypeRouter({
           "http": django_asgi_app,
           "websocket": AllowedHostsOriginValidator(
               AuthMiddlewareStack(
                   URLRouter(websocket_urlpatterns)
               )
           ),
       })
       ```

    6. Update `scripts/run-dev.sh` -- replace `manage.py runserver` with uvicorn:
       Change line `$PYTHON manage.py runserver` to:
       `$PYTHON -m uvicorn pathfinder.asgi:application --reload --host 127.0.0.1 --port 8000`
       Reason: `manage.py runserver` does NOT support WebSocket protocol upgrade. The project already uses uvicorn in Docker.

    7. Update `theme/templates/base.html` -- add htmx-ext-ws script tag AFTER htmx.min.js:
       `<script src="{% static 'js/vendor/htmx-ext-ws.min.js' %}" nonce="{{ csp_nonce }}"></script>`
  </action>

<task type="auto">
  <name>Task 2: Create ServiceConsumer with polling loop and WebSocket routing</name>
  <files>
    core/routing.py
    core/consumers.py
  </files>
  <action>
    1. Create `core/routing.py`:
       ```python
       from django.urls import path
       from core.consumers import ServiceConsumer

       websocket_urlpatterns = [
           path("ws/services/<int:service_id>/", ServiceConsumer.as_asgi()),
       ]
       ```

    2. Create `core/consumers.py` with a single `ServiceConsumer(AsyncWebsocketConsumer)`:

       **connect():**
       - Check `self.scope["user"].is_authenticated` -- if not, `await self.close()` and return
       - Extract `service_id` from `self.scope["url_route"]["kwargs"]`
       - Verify service exists (via sync_to_async query) -- if not, close
       - `await self.accept()`
       - Start polling: `self.poll_task = asyncio.create_task(self.poll_loop())`
       - Initialize `self.last_state_hash = None`

       **disconnect(close_code):**
       - Cancel self.poll_task if it exists, await it catching CancelledError

       **poll_loop():**
       - `while True:` loop with `await asyncio.sleep(3)` at the end
       - Call `get_current_state()` to fetch all relevant data
       - Compute hash via `compute_hash(state)`
       - If hash differs from `self.last_state_hash`, update hash, call `render_updates(state)`, send the HTML
       - Wrap body in try/except -- catch `asyncio.CancelledError` (re-raise) and `Exception` (log, continue)

       **get_current_state() -- wrapped in @sync_to_async:**
       - Import models inside the method to avoid AppRegistryNotReady
       - Fetch Service with select_related("ci_workflow", "ci_workflow_version", "project")
       - Fetch last 20 builds as list of values (id, status, commit_sha, commit_message, branch, created_at, started_at, duration_seconds, ci_run_url, verification_status)
       - Build stats: total count, success count, completed count, avg duration
       - Return dict with: service status, scaffold_status, ci_manifest_status, ci_workflow info, build data, stats

       **compute_hash(state):**
       - `hashlib.md5(json.dumps(state, default=str, sort_keys=True).encode()).hexdigest()`

       **render_updates(state) -- wrapped in @sync_to_async:**
       - This method will be expanded in Plan 03 when template partials are ready
       - For now, implement a placeholder that returns empty string (no OOB HTML yet)
       - The consumer structure and polling loop must be fully functional

       Use `import logging` and `logger = logging.getLogger(__name__)` for error logging in the poll loop.
  </action>
