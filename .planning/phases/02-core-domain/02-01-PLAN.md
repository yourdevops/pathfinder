<task type="auto">
  <name>Task 1: Add Project, Environment, ProjectMembership models</name>
  <files>core/models.py, core/admin.py</files>
  <action>
Add three new models to core/models.py after existing GroupMembership model:

**Project model:**
- id: BigAutoField (primary key)
- uuid: UUIDField (default uuid4, unique, db_index)
- name: CharField (max_length=20, unique) - DNS-compatible
- description: TextField (blank=True)
- env_vars: JSONField (default=list) - format: [{"key": "X", "value": "Y", "lock": false}]
- status: CharField (choices: active/inactive/archived, default='active')
- created_by: CharField (max_length=150, blank=True) - denormalized username
- created_at: DateTimeField (auto_now_add)
- updated_at: DateTimeField (auto_now)
- Meta: db_table='core_project', ordering=['name']

**Environment model:**
- id: BigAutoField (primary key)
- uuid: UUIDField (default uuid4, unique, db_index)
- project: ForeignKey to Project (CASCADE, related_name='environments')
- name: CharField (max_length=20) - DNS-compatible, unique within project
- description: TextField (blank=True)
- env_vars: JSONField (default=list) - override/extend project env_vars
- is_production: BooleanField (default=False)
- is_default: BooleanField (default=False)
- status: CharField (choices: active/inactive, default='active')
- order: IntegerField (default=10) - dev=10, staging=20, prod=30
- created_at: DateTimeField (auto_now_add)
- updated_at: DateTimeField (auto_now)
- Meta: db_table='core_environment', unique_together=['project', 'name'], ordering=['order', 'name']

**ProjectMembership model:**
- project: ForeignKey to Project (CASCADE, related_name='memberships')
- group: ForeignKey to Group (CASCADE, related_name='project_memberships')
- project_role: CharField (max_length=20, choices: owner/contributor/viewer)
- added_by: CharField (max_length=150, blank=True) - denormalized username
- created_at: DateTimeField (auto_now_add)
- Meta: db_table='core_project_membership', unique_together=['project', 'group']

Register all three models with auditlog at bottom of file.

Add Django admin configuration for all three models in core/admin.py:
- ProjectAdmin with list_display, list_filter, search_fields
- EnvironmentAdmin with list_display including project name
- ProjectMembershipAdmin with list_display showing all relationships

```html

<script>
    document.body.addEventListener('htmx:configRequest', (event) => {
        event.detail.headers['X-CSRFToken'] = document.querySelector('meta[name="csrf-token"]').content;
    });
</script>
```
